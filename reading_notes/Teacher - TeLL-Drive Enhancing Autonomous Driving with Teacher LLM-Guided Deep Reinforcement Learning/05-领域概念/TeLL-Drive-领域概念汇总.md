# TeLL-Drive 涉及领域概念汇总
## 1. 强化学习基础概念
### 1.1 深度强化学习 DRL
基于神经网络的强化学习框架，通过环境交互试错优化决策策略；原生短板为样本效率低、复杂场景泛化能力弱，本文依靠LLM引导缓解该缺陷。
### 1.2 Actor-Critic 演员评论家算法
双网络强化学习架构：Actor网络输出动作策略，Critic网络评估状态价值，兼顾策略搜索与价值估计。
### 1.3 策略蒸馏
将复杂大模型的决策分布迁移至轻量化学生网络，压缩推理开销、保留上层决策逻辑。

## 2. 大语言模型相关概念
### 2.1 大语言模型 LLM
具备自然语言推理、场景归纳、风险逻辑判断能力；原生短板推理延迟高、随机输出存在安全隐患，无法直接用于实时自动驾驶。
### 2.2 思维链 CoT
分层分步推理提示词方案，引导LLM拆解复杂驾驶场景逻辑，减少输出幻觉，输出可解释的安全驾驶策略。

## 3. 自动驾驶决策基础概念
### 3.1 POMDP 部分可观测马尔可夫决策过程
环境状态无法完全观测，仅能通过传感器获取局部观测，适配自动驾驶车载感知有限的真实场景。
### 3.2 TTCP 冲突时间
量化自车与周边车辆碰撞风险的核心指标，数值越高碰撞风险越低，作为LLM风险评估核心输入特征。

## 4. 深度表征优化概念
### 4.1 多头自注意力机制
对多源特征（LLM文本语义、DRL环境数值观测）做加权融合，挖掘跨模态关联特征，提升策略鲁棒性。
